# Freight Transportation Relational Database (FAF)

This project builds a normalized MySQL relational database for U.S. freight flows
using the Bureau of Transportation Statistics Freight Analysis Framework (FAF 5.7.1).

The goal is to demonstrate a production-style data engineering and analytics
workflow: schema design, ETL, data validation, indexing, and decision-oriented SQL.

## Project Status
🚧 Project initialized. Database schema, ETL pipeline, and analytics layers
will be added incrementally through structured pull requests.

## Data Source
- Freight Analysis Framework (FAF), Bureau of Transportation Statistics (BTS)

> Note: Raw FAF CSV files are intentionally excluded from version control due to size.
